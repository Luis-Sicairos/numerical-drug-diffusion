# Numerical Drug Diffusion Solver

## Overview

This project solves a nonlinear drug diffusion boundary value problem using several numerical methods implemented in Python.

## Methods Used

- Linear Finite Difference Method
- Nonlinear Shooting Method
- Newton's Method
- Nonlinear Finite Difference Method
- Runge-Kutta 4

## Technologies
- Python
- NumPy
- Matplotlib

## Results

The nonlinear finite difference and shooting methods produced nearly identical solutions while Newton's method converged rapidly.
